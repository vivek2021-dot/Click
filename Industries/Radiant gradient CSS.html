css
.center-object {

  z-index: 1;
}

.center-object::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: radial-gradient(circle, rgba(0,0,0,0.4) 0%, rgba(0,0,0,0) 70%);
  pointer-events: none;
  z-index: 2;
}

How this works:

  • The ::before pseudo-element adds a dark radial gradient directly over the object.

  • Since it’s attached to the object, the dark area moves wherever the object moves.

  • The pointer-events: none ensures the overlay doesn’t block clicks.

If your background is separate (like a full-screen gradient), you just keep that as normal and place this dark overlay only on the moving element.
